# Cybersecurity Notes

## Description
This repository contains personal academic notes and summaries from my 2nd year Cyber Security coursework. It covers core concepts, frameworks, and security practices that are essential for understanding information security.

## Notes Included
1. **Introduction to Cybersecurity** (`intro-cybersecurity.md`)  
   Overview of cybersecurity, its importance, and fundamental concepts.

2. **CIA Triad** (`cia-triad.md`)  
   Summary of Confidentiality, Integrity, and Availability as the core principles of information security.

3. **Types of Cybersecurity Attacks** (`types-of-attacks.md`)  
   Documenting common attacks such as malware, phishing, network attacks, and unauthorized access.

4. **OWASP Top 10** (`owasp-top10.md`)  
   Summary of the top 10 most critical web application security risks.

## Objectives
- Organize academic notes in a professional and accessible manner
- Reinforce learning by documenting key cybersecurity concepts
- Prepare a portfolio suitable for coursework review and internships

## Learning Outcomes
- Gain a solid understanding of cybersecurity fundamentals
- Familiarize with common threats, security frameworks, and best practices
- Develop skills in documenting technical information clearly

All content in this repository is part of academic coursework as a 2nd year Cyber Security student.
